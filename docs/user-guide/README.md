# ユーザーガイド

このNFL Stats Webアプリケーションは、NFLの試合情報を簡単に閲覧できるサービスです。このガイドでは、アプリケーションの各機能の使い方を平易な言葉で説明します。

## アプリケーション概要

NFL Stats Webアプリケーションは、アメリカンフットボール（NFL）の最新情報を提供するWebサイトです。試合のスコア、ニュース記事、試合スケジュール、チーム順位表、チーム情報などを一つの場所で確認できます。

### このアプリケーションでできること

- **リアルタイムの試合スコア確認**: 現在進行中の試合や終了した試合のスコアを見ることができます
- **NFLニュースの閲覧**: 最新のNFL関連ニュース記事を読むことができます
- **試合スケジュール確認**: 今週、来週、過去の週の試合予定を見ることができます
- **順位表の確認**: 各チームの成績と順位を様々な形式で確認できます
- **チーム情報の閲覧**: 全32チームの情報とロスター（選手名簿）を見ることができます

## データソース

このアプリケーションで表示されるデータは、ESPN（大手スポーツメディア）が提供するAPIから取得しています。ESPNは信頼性の高いスポーツデータプロバイダーであり、リアルタイムで正確な情報を提供しています。

### データの更新頻度

- **試合スコア**: 約60秒ごとに自動更新されます
- **ニュース記事**: 約30秒ごとに最新記事が取得されます
- **順位表・チーム情報**: 定期的に更新されます

## モバイル対応について

このアプリケーションは、スマートフォン、タブレット、デスクトップパソコンなど、あらゆるデバイスで快適に利用できるように設計されています（レスポンシブデザイン）。

### デバイスごとの特徴

- **スマートフォン**: コンパクトに最適化された表示。メニューはハンバーガーアイコン（三本線）から開きます
- **タブレット**: 中サイズの画面に適した2カラムレイアウト
- **デスクトップ**: 広い画面を活かした3カラムレイアウトで、より多くの情報を一度に表示

## ページ一覧

このアプリケーションには以下の6つのページがあります。各ページの詳細な使い方は、個別のドキュメントをご覧ください。

### 1. [Live Scores（ライブスコア）](./01-live-scores.md)

- **URL**: `/`（トップページ）
- **概要**: 現在の週の試合スコアをリアルタイムで確認できます
- **主な機能**: 週選択、シーズンタイプ切り替え、試合状態の確認

### 2. [News（ニュース）](./02-news.md)

- **URL**: `/news`
- **概要**: 最新のNFL関連ニュース記事を一覧で見ることができます
- **主な機能**: 記事カード表示、カテゴリバッジ、プレミアム記事表示

### 3. [Schedule（スケジュール）](./03-schedule.md)

- **URL**: `/schedule`
- **概要**: シーズン全体の試合スケジュールを週単位で確認できます
- **主な機能**: 週ごとのタブ切り替え、シーズンタイプ別表示

### 4. [Standings（順位表）](./04-standings.md)

- **URL**: `/standings`
- **概要**: NFLチームの順位表を様々な形式で確認できます
- **主な機能**: 全体順位、カンファレンス別、ディビジョン別の3つの表示モード

### 5. [Teams（チーム一覧）](./05-teams.md)

- **URL**: `/teams`
- **概要**: NFL全32チームの一覧を、カンファレンスとディビジョンごとに表示します
- **主な機能**: AFC/NFC切り替え、ディビジョン別カード表示

### 6. [Team Detail（チーム詳細）](./06-team-detail.md)

- **URL**: `/teams/[チームID]`
- **概要**: 選択したチームの詳細情報とロスター（選手名簿）を表示します
- **主な機能**: チームヘッダー、ポジション別ロスター表示

## ナビゲーションの使い方

### ヘッダーメニュー

アプリケーション上部には常にヘッダー（ナビゲーションバー）が表示されています。

#### デスクトップ版

デスクトップでは、ヘッダーに以下のリンクが横並びで表示されます:

- **Live** - ライブスコアページへ
- **News** - ニュースページへ
- **Schedule** - スケジュールページへ
- **Standings** - 順位表ページへ
- **Teams** - チーム一覧ページへ

現在表示しているページは、アクセントカラー（明るい色）で強調表示されます。

#### モバイル版

スマートフォンやタブレットでは、ヘッダー右上にメニューアイコン（三本線）が表示されます。このアイコンをタップすると、サイドメニューが開き、全てのページへのリンクが表示されます。

### ホームに戻る

ヘッダー左側の「NFL Stats」ロゴをクリックすると、いつでもトップページ（Live Scoresページ）に戻ることができます。

## よくある質問（FAQ）

### Q: データはどのくらいの頻度で更新されますか？

A: 試合スコアは約60秒ごと、ニュースは約30秒ごとに自動更新されます。ページを開いたままにしておけば、自動的に最新情報が表示されます。

### Q: 過去の試合結果を見ることはできますか？

A: はい。Live Scoresページで週選択機能を使うことで、過去の週の試合結果を確認できます。

### Q: スマートフォンでも使えますか？

A: はい。このアプリケーションはスマートフォン、タブレット、デスクトップなど、あらゆるデバイスに対応しています。

### Q: 特定のチームの情報だけを見たい場合は？

A: Teamsページから興味のあるチームを選択すると、そのチームの詳細ページに移動し、ロスターや情報を確認できます。

### Q: 試合の詳細なスタッツ（統計情報）は見られますか？

A: 現在のバージョンでは、スコアと基本的な試合情報のみを提供しています。詳細なスタッツ機能は将来のバージョンで追加予定です。

### Q: オフラインでも使えますか？

A: いいえ。このアプリケーションはインターネット接続が必要です。データはリアルタイムでESPN APIから取得しています。

### Q: ログインは必要ですか？

A: いいえ。このアプリケーションは全ての機能を無料で、ログインなしで利用できます。
